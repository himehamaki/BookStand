# BookStand

本棚と執筆のアプリ — 最大 10 冊の本を管理し、自分で執筆できます。

## 機能

- 📚 **本棚ビュー** — 10 冊分のスロットを木製本棚に表示。未執筆の本と執筆済みの本を一目で確認できます。
- ✏️ **執筆（編集）** — 本を選ぶと編集画面へ。表紙のタイトル・色、章（複数）、段落（複数）を自由に追加・編集・削除できます。
- 📅 **日付の自動記録** — 初めて編集した日時（執筆開始日）と最終更新日時を自動保存します。
- 📄 **エクスポート** — 本の内容をテキストファイル（`.txt`）としてダウンロードできます。
- 💾 **自動保存** — データはブラウザの `localStorage` に保存されます。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
npm run build
```

## 技術スタック

- [Vite](https://vite.dev/) + [React](https://react.dev/) + TypeScript
- スタイル: 純粋な CSS（外部 CSS フレームワーク不使用）
- データ永続化: `localStorage`
